문제5
  => 
    var :
      console.log(a); => undefined (변수 호이스팅 발생)
      var a = 1;
      var a = 2; => 재선언 가능
      a = 3; => 재할당 가능

    이러한 문제점 해결하고자 ES6 표준안에서 새로운 변수 키워드 등장

    let : 
      console.log(b); => 에러
      let b = 1;
      let b = 2; => 에러 => 재선언 금지
      b = 2; => 재할당 가능

    var : 변수 재선언 가능, 변수 재할당 가능
    let : 변수 재선언 불가능, 변수 재할당 가능

문제7
  =>
    div의 display 속성:
      1. default로 display: block으로 웹페이지의 가로 공간을 모두 차지하는 속성이다.
      2. 예를들어, 빈 div에 height만 200px으로 주고 backgroundcolor: red; 했을시 기본 margin제외 width전체가 red색으로 변한다.
      3. div, p, h1, ul, ol, table 등이 있다.
    span의 display 속성:
      1. default로 display: inline으로 컨텐츠(내용)이 끝나는 지점까지 너비를 가지는 속성
      2. 너비와 높이를 설정할 수 없다.
      3. span, a, strong, textarea등이 있다.
